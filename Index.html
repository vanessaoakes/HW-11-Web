Summary: Latitude vs. X

The purpose of the project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first puled data from the OpeanWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. Latitude. Factors we looked at included: temperature, Cloudiness, wind speed, and humidity. The site provides the source of data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
